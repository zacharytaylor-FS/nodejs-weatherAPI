# Weather Forecast API

## Node Server calling an external Weather API

-   The service we call will provide the Weather Forecast based on:
    -   Location
        -   > Zipcode q=24141(ex. Radford, Virginia)
        -   > City name q=Radford
        -   > Autoip q=auto:ip
